# Standard UI/UX Design System Guide (Upgraded for Dark Premium Cool UI)

Version: 2.0
Purpose: Complete guide for maintaining a consistent, modern, and premium dark UI across the project.

--------------------------------------------------
1. DESIGN PRINCIPLES
--------------------------------------------------

1.1 Consistency
- All components must follow the same spacing, typography, color standards, and layout.

1.2 Simplicity
- Avoid unnecessary UI complexity. Every element must have a purpose.

1.3 Accessibility
- Maintain contrast ratio of 4.5:1 minimum.
- Keyboard navigation must be supported.

1.4 Responsiveness
- UI must adapt to mobile, tablet, and desktop screens.

1.5 Premium Vibe
- Minimalist and modern aesthetic.
- Subtle depth, soft shadows, smooth transitions.

--------------------------------------------------
2. COLOR SYSTEM (Dark / Premium / Cool)
--------------------------------------------------

Primary Color: #6366F1 (Indigo) - Main actions
Secondary Color: #06B6D4 (Cyan) - Secondary actions / highlights
Accent / Success: #22C55E (Green) - Success indicators
Warning: #F59E0B (Amber) - Warnings
Error: #EF4444 (Red) - Error messages

Background: #0F172A (Dark)
Card / Section: #1E293B (Slightly lighter dark)
Navbar / Sidebar: #111827 (Darkest layer)
Text Primary: #F8FAFC (Bright white)
Text Secondary: #94A3B8 (Muted gray)
Disabled Text: #64748B (Gray)

--------------------------------------------------
3. TYPOGRAPHY (Two Premium Fonts Only)
--------------------------------------------------

Heading Font: Satoshi (Geometric, modern, premium)
Body Font: Inter (Clean, readable, neutral)

Font Weight Usage:
- 900 (Black) → Hero / Big Headings
- 700 (Bold) → Section Headings / Cards
- 500-600 (Medium) → Subheadings / Buttons
- 400 (Regular) → Body / Paragraphs

Font Sizes:
H1 / Hero: 48px (Desktop) / 36px (Mobile)
H2: 36px / 28px
H3 / Card Heading: 24px / 20px
Body Text: 16px / 14px
Small / Caption: 14px / 12px

Letter spacing:
- Headings: 0.5px - 1px
- Body: 0.2px - 0.4px

Font smoothing:
-webkit-font-smoothing: antialiased
-moz-osx-font-smoothing: grayscale

--------------------------------------------------
4. SPACING SYSTEM (8px Grid)
--------------------------------------------------

4px   → Micro spacing
8px   → Small spacing
16px  → Standard spacing
24px  → Medium spacing
32px  → Section spacing
48px  → Large section spacing

--------------------------------------------------
5. GRID & LAYOUT
--------------------------------------------------

Container Width: max 1280px
Navbar Height: 64px
Sidebar Width: 240px
Content Padding: 24px
Standard Layout: Sidebar + Navbar + Main Content + Footer

--------------------------------------------------
6. BUTTON STANDARDS
--------------------------------------------------

Primary Button:
- Background: #6366F1
- Text: #F8FAFC
- Border Radius: 12px
- Padding: 10px 16px
- Hover: #4F46E5
- Disabled: opacity 50%, cursor not-allowed

Secondary Button:
- Background: #06B6D4
- Text: #0F172A
- Hover: #0E94A2

Outline / Ghost / Danger buttons follow similar principle.

Button Typography:
- Font: Satoshi or Inter (Medium / Semi-bold)
- Letter spacing: 0.5px
- Optional Uppercase for premium feel

--------------------------------------------------
7. CARD COMPONENT
--------------------------------------------------

- Background: #1E293B
- Border Radius: 16px
- Padding: 24px
- Shadow: soft black, subtle depth
- Card structure: Header → Body → Footer (optional)

--------------------------------------------------
8. FORM DESIGN
--------------------------------------------------

- Label, input field, placeholder, validation message required
- Background: #111827
- Border: #334155
- Focus: #6366F1 (Primary Indigo)
- Placeholder: #94A3B8
- Input Height: min 40px
- Error: #EF4444
- Hover / Focus: subtle brightening

--------------------------------------------------
9. TABLE DESIGN
--------------------------------------------------

- Row Height: 48px
- Header: Bold text
- Hover row: light gray background
- Actions: icons aligned center, consistent sizing
- Font: Inter, Regular / Medium

--------------------------------------------------
10. ICON SYSTEM
--------------------------------------------------

- Size: 16px small, 20px medium, 24px default, 32px large
- Color: #F8FAFC
- Hover: #6366F1
- Alignment: vertical center with text

--------------------------------------------------
11. ANIMATION & INTERACTION
--------------------------------------------------

- Transitions: 200ms smooth
- Hover effects: scale 1.02 + shadow increase
- Loading states: skeleton loaders preferred
- Subtle depth to create premium feel

--------------------------------------------------
12. RESPONSIVE DESIGN
--------------------------------------------------

Breakpoints:
- Mobile: 640px
- Tablet: 768px
- Laptop: 1024px
- Desktop: 1280px

--------------------------------------------------
13. ACCESSIBILITY
--------------------------------------------------

- Minimum contrast ratio: 4.5:1
- Clickable element size: 40px minimum
- Keyboard navigation supported
- Use semantic HTML wherever possible

--------------------------------------------------
14. FILE STRUCTURE FOR UI COMPONENTS
--------------------------------------------------

components/
- buttons/
- cards/
- forms/
- tables/
- layout/

Each component includes:
- Component logic
- Styles
- Documentation (optional)
- Tests (optional)

--------------------------------------------------
15. UI MAINTENANCE RULES
--------------------------------------------------

- Never introduce new colors without design approval
- Reuse existing components whenever possible
- Maintain consistent spacing
- Follow design tokens strictly
- Document every new component
- Use only Satoshi (Headings) + Inter (Body) fonts consistently

--------------------------------------------------
END OF DOCUMENT
